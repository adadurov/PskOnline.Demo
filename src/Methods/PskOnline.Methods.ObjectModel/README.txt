Этот проект и пространство имен содержит общую объектную модель
сырых данных и результатов обработки по разным методикам.

* Информация о тест-сессии (прохождении 1 теста)
* Информация об обследуемых
* Стандартные физиологические данные
  (но методикам не запрещается использовать собственные форматы при необходимости)
